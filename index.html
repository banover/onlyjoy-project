How To Use

  1. +TEAM 버튼을 누른 후, bookmark할 팀을 등록한다.
  2. +입중계 버튼을 누른 후, bookmark할 youtube channel을 등록한다.
  3. 등록된 팀의 경기일정과 youtube channel live정보가 match card로 출력된다.

사용시 주의사항

제한된 API 조건으로 인한, 짧은 시간 동안의 잦은 호출은 API 오류를 발생시킬 수 있습니다.
